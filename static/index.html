進階測試

ABX 測「A 與 B 聽不聽得出差別」; X 隨機是 A 或 B, 可無限重播, 答對率過半才算聽得出. 選 EQ 檔可盲測兩組 EQ; 響度匹配避免只聽出音量差.

用上方的 Stimulus/Output mode/step 設定; 每題評 1-5 分 (1=完全在頭內, 5=明顯在頭外).

兩段聲音誰比較寬 (2AFC); 寬度用去相關雙 token 展開角呈現, 建議 stimulus 用 pink (WAV 兩側相同檔, 寬度感有限).

兩段聲音其中一段藏有目標音, QUEST 自動調目標音量找出「聽得到的最低音量」; 閾值越低代表細節解析越好.

兩段聲音哪個比較好聽 (2AFC); 建議搭配 EQ 檔比較調音.

Object Panner (HRTF sandbox)

放一到多個音源在你周圍, 拖動方位/距離/高度, 即時聽 binaural. 用通用 KEMAR HRTF (參考, 非產品 APO); 立體聲輸出, 不需 7.1. 左右穩, 高度與前後受通用 HRTF 限制.

拖兩個藍把手設 A/B 迴圈區間, 播放時只循環這段 (適合長檔剪不動的情況).

拖空白處=旋轉視角, 滾輪=縮放. 拖球=在水平面移動 (方位+距離). 高度用右邊 Height 滑桿改. 落地虛線與地板陰影顯示高度.
